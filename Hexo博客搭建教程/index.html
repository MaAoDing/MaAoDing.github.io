Hexo博客搭建教程

发表于 2022-03-06 更新于 2022-04-21 阅读次数： 本文字数： 625 阅读时长 ≈ 1 分钟

第一步 安装Node.js环境

1.node.js官网链接”https://nodejs.org/zh-cn/"

第二步 安装hexo框架

1.在命令行里输入npm install hexo-cli -g

第三步 初始化博客

1.依次在命令行里输入1 hexo init blog
2 cd blog
3 npm install

第四步 启动本地博客服务（本地进行实验）

1.在命令行里输入hexo s

第五步 部署到Github

1.Github创建一个新的仓库 YourGithubName.github.io
2.npm install hexo-deployer-git –save 在blog目录下安装git部署插件
git config –global user.email xxx
git config –global user.name “xxx”配置git
3.配置_config.yml
# Deployment
## Docs: “https://hexo.io/docs/deployment.html/"
deploy:
type: git
repo: “https://github.com/YourGithubName/YourGithubName.github.io.git/"
branch: master
4.命令行输入hexo clean以及hexo g
5.部署到Github仓库里hexo d
6.访问这个地址可以查看博客”https://YourGithubName.github.io/"
